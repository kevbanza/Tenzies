# 🎲 Tenzies Game – React Application

A small interactive Tenzies game built with React.

Tenzies is a simple dice game where the goal is to roll until all dice show the same number. Players can hold selected dice between rolls to strategically reach the winning condition.

---

## 📖 Overview

This project is a front-end game developed using React.  
It demonstrates component-based architecture, state management, and interactive UI logic.

The player:

- Rolls 10 dice
- Clicks on dice to "hold" their value
- Continues rolling until all dice have the same value and are held
- Wins when all dice match

---

## ✨ Features

- Roll 10 dice at once
- Hold/unhold individual dice
- Winning condition detection
- Game reset functionality
- Dynamic UI updates using React state
- Responsive design

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/kevbanza/Tenzies.git
cd tenzies
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm start
```

The app will run at:

```
http://localhost:3000
```

---

## 🎮 How to Play

1. Click **Roll** to roll all dice.
2. Click on a die to hold its value.
3. Keep rolling until all dice show the same number.
4. When all dice match, you win with some confetti 🎉
5. Click **New Game** to restart.
